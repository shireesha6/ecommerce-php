#  E-Commerce PHP Project

This is a simple E-Commerce website built using:

- PHP
- MySQL
- HTML
- CSS

##  Features

###  User
- Register & Login
- View Products
- Add to Cart
- Checkout
- Place Order

###  Admin
- Admin Login
- Add / Edit / Delete Products
- View Orders
- Manage Users

##  Technologies Used
- PHP (Core PHP)
- MySQL Database
- XAMPP (Local Server)

##  Project Structure
- /admin → Admin Panel
- /pages → User Pages
- /includes → Database Connection
- /css → Styling
- /images → Product Images
